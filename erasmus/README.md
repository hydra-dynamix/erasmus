# Erasmus Core

The core functionality for the Erasmus AI Context Watcher.

## Features

- Context management
- IDE integration
- Git management
- File synchronization
- Protocol management

## Installation

```bash
uv pip install -e .
```

## Usage

```bash
erasmus --help
```

## Development

```bash
# Install development dependencies
uv pip install -e ".[dev]"

# Run tests
pytest
```
